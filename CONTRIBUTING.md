# Contributing

Bestow is fully open-source.

## Did you find a bug?

If you find a bug, please [report a bug](https://github.com/redpierrot/bestow/issues) in the GitHub repo. It would be better if you can provide the steps to reproduce, environment information, and logs with enabling debug logs (using `--verbose` flag).
Before opening a bug report, please check if a similar bug has been reported already and whether it is fixed in a later release.

## Did you patched a bug?

If you have written a patch to a bug, please send a [pull request](https://github.com/redpierrot/bestow/pulls). There are no _small contributions_. Any contribution is welcomed.

## Do you find bestow is missing a feature?

If you think bestow is lacking a certain feature, please open a [feature request](https://github.com/redpierrot/bestow/issues) in the GitHub repo.

## Did you implement a feature?

If you have implemented a feature from the feature requests, please [send a PR](https://github.com/redpierrot/bestow/pulls).

## Do you have a question regarding bestow?

Please open an issue with your question.

## AI-Generated Code

Please note that bestow does not accept any AI-generated code as a contribution.

> **NOTE**: This is not because I am against the usage of AI, but for sentimental reasons.
